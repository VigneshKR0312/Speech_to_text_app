# Speech-to-Text Web App

A real-time speech-to-text web application built using React, Node.js, and Deepgram API. Users can record/upload audio and instantly convert speech into text using AI-powered transcription.

## Features

- Record voice input
- Upload audio files
- Real-time speech transcription
- Powered by Deepgram AI
- Responsive UI with React
- Secure backend API integration

## Tech Stack

Frontend:
- React.js
- CSS / Tailwind / Bootstrap (mention yours)

Backend:
- Node.js
- Express.js

API:
- Deepgram Speech Recognition API
